🧪 Unsupervised Learning – K-Means Clustering with make_blobs

A beginner-friendly Unsupervised Machine Learning project demonstrating how clustering works using synthetic data generated by make_blobs() from Scikit-learn.

This project focuses on understanding how K-Means clustering groups unlabeled data into meaningful clusters.

📌 Project Overview

Unlike supervised learning, unsupervised learning works with unlabeled data.
In this project, we:

Generate synthetic clustered data using make_blobs()
Visualize data distribution
Apply K-Means Clustering
Analyze how the algorithm groups data points
Compare real clusters vs predicted clusters

This helps build intuition for how clustering works in real-world problems like customer segmentation, anomaly detection, and pattern discovery.

🧠 Key Concepts Covered
Unsupervised Machine Learning
Clustering algorithms
K-Means algorithm intuition
Distance-based grouping (Euclidean distance)
Cluster centroids
Data visualization in 2D space
Synthetic dataset generation using make_blobs
📊 Dataset Generation (make_blobs)

We use Scikit-learn’s make_blobs() to create artificial clustered data:

Generates Gaussian-distributed clusters
Allows control over:
Number of samples
Number of clusters
Spread of clusters (cluster_std)
Feature dimensions

Example idea:

from sklearn.datasets import make_blobs

X, y = make_blobs(
    n_samples=300,
    centers=4,
    cluster_std=0.60,
    random_state=0
)

Here, y represents the true cluster labels (used only for visualization).

🤖 Algorithm Used
K-Means Clustering

K-Means works by:

Choosing initial cluster centers
Assigning each point to the nearest centroid
Updating centroids iteratively
Repeating until convergence
